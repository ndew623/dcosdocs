dcos task

管理 DC/OS 任务

  • Contribute

说明

dcos task 命令可让您管理 DC/OS 任务。

使用

dcos task [OPTIONS]

选项

名称 说明
-h, --help 打印用法。
--info 打印该子命令的简短描述。
--version 打印版本信息。
