群集

使用群集菜单

  • Contribute

从 UI，您可以查看单个群集或 链接群集 统计信息的摘要。

概述

要查看群集配置详细信息的显示，请单击 群集 > 概述。

图 1 - 群集概述

此视图显示一般详情、Apache® Mesos® 详情、Marathon™ 详情，以及有关 Bootstrap 配置的信息。

您可以从右上角的下拉菜单中看到群集的 IP 地址。

图 2 - 群集菜单

链接群集 Enterprise

多群集用于隔离（例如，测试与生产）、适应地理分布等等情境。DC/OS 多群集操作可让操作员和用户轻松管理和访问多个 DC/OS 群集。

要查看链接群集详细信息的显示，请单击 链接群集。

图 3 - 链接群集

有关链接群集的详细信息，请参阅 多个群集 文档。
