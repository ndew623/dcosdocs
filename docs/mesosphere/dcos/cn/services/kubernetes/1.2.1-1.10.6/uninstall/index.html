卸载 DC/OS Kubernetes

  • Contribute
  • Discuss
  • Feedback

从 DC/OS CLI 输入 dcos package uninstall --app-id=kubernetes kubernetes。

例如，要卸载名为 kubernetes-dev 的 Kubernetes 实例，则运行：

dcos package uninstall --app-id=kubernetes-dev kubernetes
