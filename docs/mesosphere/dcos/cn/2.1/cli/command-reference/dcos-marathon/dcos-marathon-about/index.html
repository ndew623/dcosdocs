dcos marathon about

为 DC/OS Marathon 显示 info.json 文件

  • Contribute

说明

dcos marathon about 命令将为 DC/OS Marathon 打印 info.json 文件。

使用

dcos marathon about

选项

名称 说明
--help 显示此消息并退出。
