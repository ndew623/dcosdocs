Cassandra 2.3.0-3.0.16

  • Contribute

DC/OS Apache Cassandra 是一项自动化服务，可轻松部署和管理 DC/OS 上的 Apache Cassandra。Apache Cassandra 是一个分布式 NoSQL 数据库，可在数据中心中提供高可用性、容错性和可扩展性。

有关 Apache Cassandra 的详细信息，请参阅 Apache Cassandra 文档。

特征

  • 易于安装
  • Cassandra 节点可轻松水平扩展
  • 轻松备份和恢复丢失的数据
  • 多数据中心复制支持
