Granting access to the DC/OS UI

  • Contribute

By default, a new user has no permissions and cannot view the DC/OS UI. You must grant users and groups access to each screen of the web interface.

IMPORTANT: Only superusers can view the Dashboard, Nodes, and Cluster -> Overview tabs.
