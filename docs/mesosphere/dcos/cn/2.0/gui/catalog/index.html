目录

“目录”页面显示在 DC/OS 上运行的所有服务

  • Contribute

Catalog 页面显示所有可用的 DC/OS™ 服务。只需单击即可在 DC/OS Catalog 中安装包。可以使用默认设置安装包，或直接在用户界面中自定义安装。

包类型

D2iQ® 提供两种服务包：认证和社区。

认证包

已认证的包经过 D2iQ 验证，可与 DC/OS 互操作。

图 1 - 认证包

支持这些认证包的文档可在 [DC/OS 服务文档页面] 找到(/dcosdocs/mesosphere/dcos/cn/services/)。

社区包

社区包由 D2iQ 合作伙伴和开源社区的成员贡献。它们没有被 D2iQ 验证为可与 DC/OS 互操作。但是，它们可以提供在认证 Catalog 中不可获得的许多功能。新包为定期提供。其中有些包在 DC/OS 服务文档页面 提供 Mesosphere 特定的文档。

图 2 - 社区包

搜索 Catalog

您可以在页面顶部的搜索框中搜索整个 Catalog。

图 3 - 搜索框

从 Catalog 安装

您可以运行您创建的 DC/OS 服务，或从 Catalog 安装包。创建的服务和从 Catalog 安装的服务在运行时都会显示在 DC/OS UI 的 服务 选项卡上。

安装服务的快速简单方式是从 Catalog 页面安装。

  1. 导航到 DC/OS UI 中的 Catalog 选项卡。

    图 4 - Catalog 选项卡

  2. 单击一个包。

  3. 查看安装前说明（若有）。确保您的群集符合列出的要求。单击 审查并运行。

  4. 根据需要编辑配置。

  5. 再次单击 查看并运行，检查您的配置。当您满意时，单击 运行服务。

  6. 从 服务 选项卡查看部署的服务。

    图 5 - 服务部署

有关从 Catalog 安装、配置和部署服务的更多信息，请参阅 部署服务文档。
