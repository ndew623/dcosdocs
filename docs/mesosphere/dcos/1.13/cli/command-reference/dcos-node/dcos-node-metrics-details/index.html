dcos node metrics details

Displaying the details of Mesos agent nodes

  • Contribute
  • Feedback

Description

The dcos node metrics details command prints a table of all metrics for the agent node specified by .

Usage

dcos node metrics details <mesos-id>  [--json]

Options

Name Description
--help, h Displays usage.
--json Displays JSON-formatted data.
--mesos-id=<mesos-id> The agent ID of a node.

Parent command

Command Description
dcos node View DC/OS node information.
