dcos node

显示 DC/OS 节点信息

  • Contribute

警告：从 DC/OS 1.13 开始，使用 dcos node 命令列出节点已被弃用。要列出 dcos 节点，请使用 dcos node list 命令。

说明

dcos node 命令允许您查看 DC/OS 节点信息。

使用

dcos node [OPTION]

选项

名称 说明
--help, h 显示用法。
--info 显示该子命令的简短描述。
--json 显示 JSON 格式的数据。
--version 显示版本信息。

示例

dcos node
Getting the list of nodes from `dcos node` is deprecated. Please use `dcos node list`.
