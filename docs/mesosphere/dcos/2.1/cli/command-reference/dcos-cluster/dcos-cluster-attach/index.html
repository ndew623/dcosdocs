dcos cluster attach

Attaching the CLI to a connected or linked cluster

  • Contribute
  • Feedback

Description

The dcos cluster attach command will attach the CLI to a connected or linked cluster. When you run the dcos cluster setup command, the cluster is automatically attached.

Usage

dcos cluster attach <cluster> [flags]

Options

Name Description
--help, h Display usage.

Positional arguments

Name Description
<cluster> ID of cluster (required)

Examples

dcos cluster list
